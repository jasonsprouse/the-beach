### Installation Guide

```bash
# 1. Clone repositories
git clone https://github.com/jasonsprouse/the-beach

# 2. Install dependencies

# Edit .env with your settings

# 4. Build applications
npm run build

# 5. Run health check

# 7. Verify

```
