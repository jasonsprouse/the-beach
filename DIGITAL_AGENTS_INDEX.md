### Lit Compute Network
**Definition:** Distributed CPU processing network for Lit Protocol encryption operations

**How It Works:**
1. User submits encryption job (via Y8 App)
2. Job enters queue (managed by NPE agents)
3. Available node picks up job
4. Node processes encryption using Lit Protocol
5. Node uploads result to IPFS
6. Smart contract releases payment
7. User downloads encrypted result

**Earnings Potential:** $1,500-4,500/month per node (8-core CPU)
